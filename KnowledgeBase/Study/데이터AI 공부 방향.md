데이터/AI 쪽 관심 있으시다면 **자격증 로드맵을 “기초 → 데이터 → 클라우드 AI 특화”** 식으로 가시면 좋아요.  
정리해 드리면:

---

## 📊 1단계: 데이터 기초 다지기

- **ADsP (데이터분석 준전문가)**
    
    - 난이도: ★★☆☆☆
        
    - 데이터 기초 이론 + 간단한 분석 방법론.
        
    - 비전공자도 시작하기 좋아요.
        
- 👉 이걸로 데이터/AI 기본 개념을 잡고, 이력서에도 “데이터 기초 자격 보유” 라고 적을 수 있습니다.
    

---

## ☁️ 2단계: 클라우드 입문

- **AWS Cloud Practitioner** 또는 **Azure AZ-900**
    
    - 클라우드 기초 서비스, AI/ML 서비스가 어떻게 제공되는지 맛보기.
        
    - AI 쪽을 본격적으로 하기 전에, “클라우드는 이렇게 돌아가는구나” 감 잡는 단계.
        

---

## 🤖 3단계: 클라우드 + 데이터/AI 특화

- **AWS Solutions Architect – Associate (SAA-C03)**
    
    - 범용성 최고, 필수 기반.
        
- 그다음 선택적으로:
    
    - **AWS Machine Learning Specialty** (AI/ML 특화)
        
    - **Google Professional Data Engineer** (빅데이터·AI에 최적화, 난이도 높음)
        

---

## 📌 추천 로드맵 (데이터/AI 관심 + 기초 단계인 경우)

1. **ADsP** (데이터 기본기 다지기)
    
2. **AWS Cloud Practitioner (or Azure AZ-900)** (클라우드 기본기 다지기)
    
3. **AWS SAA-C03** (실무에서 바로 쓰이는 자격)
    
4. 관심 생기면 → **AWS ML Specialty** 또는 **GCP Data Engineer**
    

---

👉 이렇게 하면

- **ADsP**로 데이터 기본기,
    
- **AWS/Azure 입문**으로 클라우드 기본기,
    
- **AWS SAA**로 실무 클라우드 역량,
    
- 그다음에 **AI/ML 특화 자격증**까지 가는 그림이 완성돼요.
