# ✅ 1) **Grammar in Use – Intermediate에는 영단어(보카) 없음.**

이 책은 **문법 전용**이고,  
단어 학습 파트는 **거의 없다.**

그런데 **너는 별도 영어 단어책(보카) 필요 없다.**

왜냐?

### 🔥 개발자에게 필요한 단어는 95%가 **기술 영어**라서

TOEIC 단어 / 수능 영단어 / 회화 단어 = **거의 의미 없음**.

개발자에게 필요한 단어는 아래 같은 것들:

- scalable
    
- throughput
    
- concurrency
    
- batch
    
- distributed
    
- instance
    
- endpoint
    
- latency
    
- redundancy
    
- traffic
    
- fallback
    
- refactor
    
- inference
    
- token
    
- embedding
    
- orchestration
    
- failover
    
- pipeline
    

이건 "보카책"으로 공부하는 단어가 아님.  
**기술 문서(AWS·GCP·OpenAI·React·Node) 보면 저절로 배우게 되는 단어들.**

📌 그래서 **일반 보카책은 시간 낭비**임.  
→ 대신 "기술 문서 + 기술 유튜브"에서 자연스럽게 익히는 게 훨씬 효율 높음.
