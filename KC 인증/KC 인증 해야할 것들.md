좋아, 그러면 **AI 향수 추천 키오스크(발향 + 조제 기능)** 기준으로  
**KC 인증을 정확하게 어떤 유형으로 받아야 하는지**,  
그리고 **KTC/KTL/KTR 중 어디가 맞는지**,  
**시험 항목이 무엇인지**,  
**2월까지 가능할지 현실적인 타임라인**까지 “실무 버전”으로 정리해줄게.

# ✅ 1. 제품 정의(너무 중요)

키오스크는 크게 2개의 기능을 가짐:

### **① 발향 기능 (향 분사/시연 장치)**

- 전기 구동(모터, 펌프, 팬 등)
    
- 전자제어(PCB)
    
- 사용자가 가까이에서 향을 맡음 → 인체접촉 제품군
    
- 압력/노즐 사용 가능성 있음
    

### **② 조제 기능 (향수 원액 + 디퓨저 용액 자동 배합 → 10ml 샘플 제작)**

- 액체 취급(화학물질 포함)
    
- 펌프/밸브/튜빙 포함
    
- 혼합 후 용기 충진(소형 filling system)
    
- 사용자가 바로 수령하여 사용 → 소비재, 인체 접촉
    

### **③ 키오스크 본체 (터치패널 + 컴퓨터 + 네트워크)**

- 모니터, PC, 전기부품
    
- 인터넷/블루투스/Wi-Fi 가능성 있음  
    → 전파 인증 가능성 있음
    

즉, 이 제품은 **전기용품 + 생활용품 + 화장품 관련 기계 장비 + 무선 기능**이 합쳐진 복합 구조야.

---

# ✅ 2. KC 인증 “어떤 유형”을 받아야 하는지 (정확판)

## 🔶 **① 전기·전자 KC 인증 (전기용품 및 생활용품 안전관리법)**

해당됨.

이유:

- 키오스크는 전원을 사용하는 전기기기
    
- 펌프·모터·팬 등 전기부품 포함
    
- 사용자 접촉 환경(매장 상업환경)
    

### **➡ 적용되는 KC 유형**

- 위험도 중간~높음 → **안전확인(안전확인시험)** 가능성이 매우 높음
    
- 일부 구성품(어댑터, 파워서플라이)은 **별도 KC 인증 필수**
    

**시험 기관:**  
KTC / KTL / KTR 중 선택 가능.

---

## 🔶 **② 전자파 KC(EMC: 방사/내성)**

해당됨.

이유:

- 터치PC, 메인보드, 디스플레이
    
- Wi-Fi/BLE 기능 들어갈 경우 전파법 대상
    

### **➡ 적용되는 KC 유형**

- **전자파적합성(EMC) 시험**
    
- **무선기기 적합성(RF) 시험** – 무선 모듈 포함 시
    

**시험 기관:**  
KTC, KTL, KTR 모두 가능.

---

## 🔶 **③ 화학제품 / 향수 조제기능 관련 KC**

여기서 중요한 포인트.

향수는 법적으로 **화장품**으로 분류됨.  
하지만 키오스크는 화장품을 제조·판매하는 “기계”일 뿐,  
기계 자체는 **KC 화장품 인증 대상이 아님**.

### **➡ 대신 다음 2가지가 필요함**

1. **조제 과정에서 인체 유해물질이 노출되지 않도록 → 생활용품 기계류 KC 시험 항목 적용**
    
    - 누액, 과열, 전기안전, 사용 중 기계적 위험성 등
        
2. **최종 출력물(10ml 향수)** 자체는
    
    - **화장품 제조업 신고 + 책임판매업 신고** 영역 (KC 아님)
        
    - 보건복지부 / 화장품법 기준
        
    - 한국 판매 시 필수.
        

---

## 🔶 **④ 사용자 접촉 장비 관련 KC – 생활용품 안전 기준 일부 적용**

(기계가 액체를 다루고 소비자가 향을 흡입/접촉함)

KC 생활용품 항목 중 다음이 적용됨:

- 누액 방지
    
- 재질 안전성(원료 저장 팩, 튜브 등)
    
- 화학제품 접촉 부품의 내식성 및 안전성
    
- 고장 시 과열, 감전, 화재 위험 등의 안전시험
    

---

## 🔶 **최종 결론: 필요한 KC 인증 리스트**

### **1) KC 안전확인 (전기·전자) – 본체 전체**

### **2) KC EMC(전자파) – 반드시 필요**

### **3) KC RF(무선) – Wi-Fi/BLE 모듈 있을 때**

### **4) 화장품 조제기기 관련 생활용품 안전성 평가 – 필요 항목만 시험**

### **5) 전원 어댑터는 개별 KC 인증 필수**

# ✅ 3. 시험 항목(실제 시험 리스트)

KC 안전확인 기준(전기·전자 + 기계류 혼합)에서 필요한 시험:

### 🔧 **전기 안전 시험**

- 누설전류
    
- 절연저항
    
- 내전압
    
- 온도상승
    
- 과부하 시험
    
- 단락 보호
    
- 접지 연속성
    
- 소손/발연 여부
    

### 🔧 **기계적 안정성**

- 구조적 강도
    
- 동작 중 위험요소(움직이는 부분)
    
- 안정성(넘어짐 방지)
    

### 🔧 **기능 안전 (액체펌핑/조제 관련)**

- 누액 테스트
    
- 용기 충진 시 과충진 방지
    
- 배관·튜브 내구성
    
- 내부 화학물질(향수)과의 접촉 안전성
    

### 🔧 **EMC**

- 방사성(EMI)
    
- 내성(EMS)
    
- ESD 정전기 시험
    
- 전도성 노이즈 시험
    

### 🔧 **무선 모듈 RF**

- 출력
    
- 주파수
    
- 점유대역폭
    
- 스퓨리어스 방사

# ✅ 4. 인증기관 추천 (KTC / KTL / KTR 중)

### ✔ **가장 적합한 기관: KTC (한국기계전기전자시험연구원)**

**이유**

- 전자장비 + 기계장비(펌프, 모터) 복합제품 경험 多
    
- KIOSK·소형 산업기기 시험 경험 풍부
    
- EMC + 안전시험 모두 한곳에서 가능
    

→ **너희 제품 같은 복합기기에 가장 강함**

# ✅ 5. 예상 비용 + 기간 (현실적인 수치)

### 💰 **비용 예상**

- KC 안전확인: **250~400만 원**
    
- EMC(전자파): **200~350만 원**
    
- 무선(RF): **300~500만 원** (모듈에 따라)
    
- 생활용품 안전성 시험: **100~200만 원**
    
- 어댑터는 별도 구매: KC 인증된 정격제품 사용하면 비용 없음
    

**총합:**  
➡ **650만 ~ 1,200만 원 수준**

---

### ⏱ **기간(2월까지 가능 여부)**

- 접수 전 서류준비: 1~2주
    
- 시험기간: 3~5주
    
- 무선 있을 경우: 추가 +2~3주
    
- 전체 소요: **6~8주**
    

### **➡ 지금 바로 시작하면 2월 인증은 ‘빠르게 진행했을 때 딱 맞는 수준’**

(특히 무선 모듈 있으면 촉박하지만 가능)
