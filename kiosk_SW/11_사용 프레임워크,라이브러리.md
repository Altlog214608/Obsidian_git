# 프로젝트에서 사용된 주요 프레임워크/라이브러리 분석

## 1. 핵심 프레임워크
- **Electron**: 데스크톱 애플리케이션 프레임워크
- **React**: UI 라이브러리
- **TypeScript**: 정적 타입 시스템

## 2. 빌드/개발 도구
- **Vite**: 빌드 도구 및 개발 서버
- **electron-vite**: Electron + Vite 통합
- **electron-builder**: 애플리케이션 패키징/배포

## 3. 스타일링
- **Tailwind CSS**: 유틸리티 기반 CSS 프레임워크

## 4. 상태 관리/라우팅
- **React Router DOM**: 라우팅 (`react-router-dom`)

## 5. UI/애니메이션 라이브러리
- **Framer Motion**: 애니메이션 (`framer-motion`)
- **GSAP**: 고급 애니메이션 (`gsap`)
- **React Spring**: 물리 기반 애니메이션 (`@react-spring/web`)
- **OGL**: WebGL 그래픽스 라이브러리 (현재 누락)

## 6. 데이터 시각화
- **Recharts**: 차트 라이브러리
- **D3**: 데이터 시각화 유틸리티
  - `d3-color`
  - `d3-interpolate`
  - `d3-scale`
  - `d3-shape`

## 7. 아이콘/디자인 시스템
- **Myna UI Icons**: 아이콘 세트 (`@mynaui/icons`, `@mynaui/icons-react`)
- **Phosphor Icons**: 아이콘 세트 (`@phosphor-icons/react`)

## 8. 폰트
- **Fontsource**: 웹폰트
  - `@fontsource/castoro`
  - `@fontsource/jetbrains-mono`
  - `@fontsource/noto-sans-kr`
  - `@fontsource/playwrite-us-trad`

## 9. 분석/모니터링
- **Mixpanel**: 사용자 행동 분석 (`mixpanel-browser`)

## 10. 개발 도구/유틸리티
- **ESLint**: 코드 린팅
- **Prettier**: 코드 포맷팅
- **electron-toolkit**: Electron 유틸리티
  - `@electron-toolkit/preload`
  - `@electron-toolkit/utils`

## 11. 기타 UI 컴포넌트/효과
- **React Liquid Gauge**: 게이지 컴포넌트
- **React Wavify**: 웨이브 효과
- **Mesh Gradient**: 그라데이션 효과 (`@mesh-gradient/react`)

이 리스트는 package.json의 dependencies와 devDependencies를 기반으로 작성되었습니다. 각 라이브러리의 구체적인 버전이나 설정이 필요하시다면 알려주세요.
