|                | ② **적용 기준 규격:** IEC 60601-1-2를 기반으로 하는 **'의료기기 전자파 안전에 관한 공통 기준규격'** 준수를 목표로 설정하고, 시험 항목에 전자파 방해(EMI)와 내성(EMS) 시험을 모두 포함하도록 확정.                                  |

|                | ① **시험 유형 확정:** '제품시험검사' 및 '전자파장해(EMI) 시험' 항목을 명확히 지정하여 신청서에 반영하고, 시험 목적으로 '제조품목허가/인증신청'을 최종 확인.                                                                 |
| **2025.03.14** | **1. 전자파 방해(EMI) 저감 대책 적용 및 시스템 안정화**                                                                                                                            |
|                | (1) 전자파 방출(Emission) 원인 분석 및 개선                                                                                                                                  |
|                | ① **스위칭 노이즈 관리:** 발향 장치 구동부(펌프/밸브)의 PWM(Pulse Width Modulation) 제어 회로에서 발생하는 스위칭 노이즈를 저감하기 위해, 전원 라인에 고성능 필터링 회로(LC 필터)를 추가 삽입하고 효과 검증.                          |
|                | ② **접지(Grounding) 및 차폐(Shielding) 최적화:** PCB 레이아웃 상의 접지 플레인(Ground Plane)을 강화하고, 시스템 케이싱 내부에 전도성 차폐 재료를 적용하여 복사 방출(Radiated Emission) 레벨을 허용 기준치 이하로 유지하는 조치 완료. |
|                | (2) 시스템 유효성 유지                                                                                                                                                   |
|                | ① **EMC 환경 조건에서의 동작 확인:** 외부 노이즈 환경(예: R/F Field)에서도 후각 검사 소프트웨어의 발향 제어 타이밍 및 피검자 응답 데이터 처리의 오류가 발생하지 않도록 Fail-Safe 로직 구현.                                       |
| **2025.05.16** | **1. 전자파 시험검사 최종 성적서 수령 및 규격 적합성 판정**                                                                                                                            |

|                | ② **전원 안전성 확보:** IEC 60601-1의 일반 요구사항 중 4.10.2항(공급 전원 조건)에 대한 시험 결과 역시 적합으로 판정되어, 본 제품의 전기적 안전성과 전자파 안전성을 동시 확보.                                                 |

|**2025.01.10**|**1. 의료기기 전자파 안전 기준 규격(EMC Standard) 심층 분석 및 내부 점검**|

||② **내부 예비 점검 수행:** 개발 완료된 '후각 검사기구(모델명: OD-1/DS-ODTD-01)'의 시제품에 대해, 내부 스위칭 노이즈 및 외부 노이즈에 대한 SW의 오작동 여부를 시뮬레이션 환경에서 예비 테스트.|
|**2025.01.20**|**1. EMC 시험 대상 제품 준비 및 위험 관리 계획 수립**|
||(1) 시험용 시료(Test Sample) 및 조건 정의|
||① **시료 상세 정보 확정:** 시험에 제출할 제품 모델명(OD-1/DS-ODTD-01), 제조번호(DS-ODTD-01-2503001), 제조 일자(2025-01)를 최종 확정하고 시험기관 제출용으로 준비 완료.|
||② **시험 운용 모드 정의:** 시험 중 모든 핵심 기능(발향 제어, 데이터 통신, 터치 입력)이 **연속적으로 동작**하는 **'최대 부하 운용 모드'**를 설정하여, 최악의 시나리오(Worst-Case Scenario)에서 전자파 적합성을 검증할 수 있도록 계획.|
||(2) 전원 공급 조건 및 안전성 사전 검토|
||① **공급 전원 조건(IEC 60601-1 4.10.2항) 확인:** 제품의 정격 입력 전압 및 주파수 환경에서의 SW/HW 동작 안정성을 사전에 체크하고, 시험 성적서 결과에서 '적합' 판정을 받을 수 있도록 초기 설계 오류 여부 최종 점검.|
|**2025.01.31**|**1. 공인 시험기관(KTC) 전자파 적합성 시험 신청 및 제출 완료**|
||(1) 공식 시험 검사 신청서 제출|
||① **KTC 시험 접수 완료:** (재)한국기계전기전자시험연구원(KTC)에 '의료기기 시험검사신청서'를 공식 제출 (접수번호: 전파의료2025-00005)하고, 시험 목적으로 **'제조품목허가/인증신청'**을 명시.|
||② **제출 자료 적합성 확인:** 신청서에 기재된 모든 정보(업소명, 소재지, 제품명, 모델명, 제조국)가 사업자등록 및 제조원 정보와 일치하는지 최종 확인하고 시험 담당자에게 자료 전달 완료.|
