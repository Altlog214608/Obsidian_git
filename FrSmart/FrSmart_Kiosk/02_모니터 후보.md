## 🖥️ 권장 PC 사양 (현대백화점 키오스크용)

### 1. CPU

- **Intel 12~13세대 i5 이상** (저전력/산업용 라인 가능)  
    → 웹/앱 UI + 간단한 ML 모델 실행에는 충분
    
- GPU 가속 필요하면 **i7 + 내장 Iris Xe** 또는 **작은 외장 GPU(NVIDIA T600, GTX 1650급)** 고려
    

### 2. 메모리 (RAM)

- **16GB 기본 / 32GB 권장**  
    → 다중 디스플레이, AI 모델 로컬 실행, 장기간 무중단 운영 시 안정성 확보
    

### 3. 저장장치

- **NVMe SSD 512GB 이상**  
    → OS + 프로그램 + 로그/사용 데이터 여유 있게 확보
    
- 백업/로그 전송은 NAS·클라우드 연동 추천
    

### 4. 그래픽

- **FHD(1920×1080) 기본 / QHD~4K(선택)**  
    → 27″ 이상 화면은 FHD 이상이 깔끔, 32″라면 QHD 권장
    
- 내장 GPU (Intel Iris Xe, AMD Radeon iGPU)로 충분
    
- 영상/3D 요소 많으면 **외장 GPU** 고려
    

### 5. 네트워크 & 포트

- **LAN + Wi-Fi** 필수, (추가) LTE 모듈 가능
    
- **USB 포트 ≥6개** (향 제어 MCU, 보조 장치 연결)
    
- **시리얼포트(COM)** 있으면 산업용 제어와 직결 가능
    

### 6. OS

- **Windows 11 IoT Enterprise (LTSC)** → 키오스크 락다운/장기지원에 최적
    
- 또는 **Ubuntu LTS** → 비용절감 + 경량 컨테이너 환경

## 📌 요약 추천

## 🖥️ 박스 PC (Box PC)

- **형태**: 독립된 외부 박스형 PC (소형 데스크탑 비슷, 팬리스 산업용도 있음)
    
- **설치**: 키오스크 본체 내부나 후면에 장착 → 디스플레이와 HDMI/DP, USB 케이블로 연결
    
- **장점**
    
    - 확장성 좋음 (USB 포트, LAN, 시리얼포트 등 많음)
        
    - 발열·전원부 용량 여유 있어서 안정적
        
    - 교체/업그레이드가 쉬움
        
- **단점**
    
    - 케이블 배선 필요 (깔끔함 ↓)
        
    - 키오스크 내부 공간 차지
        

---

## 🖥️ OPS / SDM 타입 PC

- **OPS (Open Pluggable Specification)**
    
    - 인텔이 만든 규격, **디스플레이 뒷면 슬롯에 꽂는 모듈형 PC**
        
    - 일체형처럼 보이지만 실제로는 PC 모듈을 디스플레이에 삽입하는 구조
        
- **SDM (Smart Display Module)**
    
    - OPS 후속 규격, 더 얇고 고성능 지원
        
- **장점**
    
    - **디스플레이 뒤에 바로 삽입 → 케이블 최소화, 외관 깔끔**
        
    - 유지보수: 모듈만 교체 가능
        
- **단점**
    
    - 확장성 제한 (USB 포트·LAN 적음)
        
    - 발열 여유 적음 (고성능/24시간 운용에는 불리)
        
    - 디스플레이가 OPS/SDM 슬롯을 지원해야만 사용 가능
        

---

## ✅ 정리

- **박스PC**: 안정성·확장성 최우선 → 산업용/장시간 운용 적합
    
- **OPS/SDM PC**: 디자인·배선 깔끔 → 매장 디스플레이용, 공간 절약
    

👉 쉽게 말해,

- **“진짜 일체형 PC 모니터”** = PC 부품이 모니터 안에 다 들어있는 것 (주로 올인원 PC)
    
- **“OPS/SDM”** = 모니터 뒤에 **교체 가능한 PC 모듈을 끼우는 방식**
    
- **“박스PC”** = 완전 별도의 미니PC를 옆에 두고 케이블 연결하는 방식
    

---

현대백화점에 설치한다면,

- **고객 응대용 깔끔한 외형**: OPS/SDM → 디자인 우위
    
- **AI/향 분사 제어 + 장시간 안정성**: 박스 PC → 기능·안정성 우위
