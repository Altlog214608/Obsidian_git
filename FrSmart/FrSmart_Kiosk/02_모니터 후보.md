## 🖥️ 권장 PC 사양 (현대백화점 키오스크용)

### 1. CPU

- **Intel 12~13세대 i5 이상** (저전력/산업용 라인 가능)  
    → 웹/앱 UI + 간단한 ML 모델 실행에는 충분
    
- GPU 가속 필요하면 **i7 + 내장 Iris Xe** 또는 **작은 외장 GPU(NVIDIA T600, GTX 1650급)** 고려
    

### 2. 메모리 (RAM)

- **16GB 기본 / 32GB 권장**  
    → 다중 디스플레이, AI 모델 로컬 실행, 장기간 무중단 운영 시 안정성 확보
    

### 3. 저장장치

- **NVMe SSD 512GB 이상**  
    → OS + 프로그램 + 로그/사용 데이터 여유 있게 확보
    
- 백업/로그 전송은 NAS·클라우드 연동 추천
    

### 4. 그래픽

- **FHD(1920×1080) 기본 / QHD~4K(선택)**  
    → 27″ 이상 화면은 FHD 이상이 깔끔, 32″라면 QHD 권장
    
- 내장 GPU (Intel Iris Xe, AMD Radeon iGPU)로 충분
    
- 영상/3D 요소 많으면 **외장 GPU** 고려
    

### 5. 네트워크 & 포트

- **LAN + Wi-Fi** 필수, (추가) LTE 모듈 가능
    
- **USB 포트 ≥6개** (향 제어 MCU, 보조 장치 연결)
    
- **시리얼포트(COM)** 있으면 산업용 제어와 직결 가능
    

### 6. OS

- **Windows 11 IoT Enterprise (LTSC)** → 키오스크 락다운/장기지원에 최적
    
- 또는 **Ubuntu LTS** → 비용절감 + 경량 컨테이너 환경
